LFMF-SmoothEarth v1 (04.06.21)

MATLAB/Octave implementation of LFMF-SmoothEarth

GENERAL NOTES
--------------

 tl_LFMFSmoothEarth.m	         - MATLAB/Octave implementation of LFMF-SmoothEarth computing LFMF smooth earth propagation prediction
			           Translated and adapted to MATLAB/Octave starting from the original C++ code
                                   from NTIA, USA

 
 validate_LFMFSmoothEarth.m      - MATLAB script used to validate the function tl_LFMFSmoothEarth.m against the reference data produced by the 
                                   original C++ implementation and available in ValidationExampleLFMFSmoothEarth.csv

UPDATES AND FIXES
-----------------
